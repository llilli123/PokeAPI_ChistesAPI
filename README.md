Descripción del Proyecto
Este proyecto consiste en el desarrollo de una página web utilizando **HTML, CSS y JavaScript** que consume dos APIs públicas para mostrar información dinámica al usuario.

La aplicación permite:

- Buscar información sobre Pokémon utilizando la **PokéAPI**.
- Mostrar chistes aleatorios utilizando la **JokeAPI**.
- Presentar la información de forma organizada

El objetivo principal es aplicar los conocimientos sobre consumo de APIs mediante el uso de fetch().

Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript 
- Fetch API
- APIs REST públicas

Uso del Proyecto

1. Abrir el archivo `index.html` en el navegador.
2. Seleccionar la sección deseada.
3. Buscar un Pokémon o generar un chiste.

Uso de APIs

PokéAPI
Se utiliza para obtener datos detallados sobre Pokémon mediante peticiones HTTP usando `fetch()`.

JokeAPI
Se utiliza para obtener chistes aleatorios con filtros de seguridad.
